# Online Quiz Web App

A beginner-friendly Python Flask web application for taking online quizzes.  
The app includes **login, registration, password authentication**, and **multi-subject quiz functionality**.

## Features
- User registration and login system
- Password-protected quiz
- Multi-subject questions (English, Aptitude, Python)
- Shows user score at the end of the quiz
- Browser-based, accessible from any device (with deployment)

## Technologies Used
- Python
- Flask (Web Framework)
- HTML (Embedded in Flask)
